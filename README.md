:pushpin: **How to use**

1. Clone this repo to your machine

```bash
$ git clone git@github.com:phurinutwara/dotfiles.git --recursive
```

2. Go to dotfiles directory and execute the installer

```bash
$ cd ~/dotfiles && ./install
```
